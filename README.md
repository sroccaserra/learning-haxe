Learning Haxe, and the Heaps library.

Prerequisites (see links bellow):

- Install the Haxe programming language.
- Install the HashLink VM.
- Install the Heaps library.

Build with:

```shell
$ haxe compile.xml
```

Run with:

```shell
$ hl hello.hl
```
